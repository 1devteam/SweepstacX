# dupes

The `dupes` command is a legacy command that has been superseded by the **Duplication Analyzer** integrated into the main `scan` command in v0.8.0.

For comprehensive code duplication analysis, please use:

\`\`\`bash
sweepstacx scan
sweepstacx report
\`\`\`

The duplication metrics and issues will be included in the standard report.

## Legacy Usage (Deprecated)

\`\`\`bash
# This command is deprecated in v0.8.0
sweepstacx dupes [options]
\`\`\`
