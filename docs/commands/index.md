# Commands Overview

SweepstacX provides 17 powerful commands to manage your code quality lifecycle.

| Command | Description | Core Functionality |
| :--- | :--- | :--- |
| `scan` | Performs a full analysis of the codebase (dead code, imports, deps). | **Analysis** |
| `report` | Generates a detailed report from the last scan. | **Reporting** |
| `patch` | Generates or applies patches to fix issues (e.g., unused imports). | **Auto-Fix** |
| `check` | Runs a check against configured quality gates (e.g., max complexity). | **CI/CD** |
| `insights` | Displays high-level metrics and trends. | **Metrics** |
| `init` | Initializes a new `.sweeperc.json` configuration file. | **Setup** |
| `watch` | Runs the scanner in watch mode for continuous feedback. | **Developer Experience** |
| `html` | Generates a full HTML dashboard report. | **Reporting** |
| `export` | Exports the report to CSV or PDF format. | **Reporting** |
| `diff` | Compares the current scan against a previous baseline. | **Trend Analysis** |
| `security` | Runs the security vulnerability scanner. | **Security** |
| `complexity` | Analyzes and reports on code complexity metrics. | **Analysis** |
| `benchmark` | Measures the performance of the SweepstacX scanner. | **Performance** |
| `trends` | Visualizes code quality trends over time. | **Metrics** |
| `plugins` | Manages and lists installed plugins. | **Extensibility** |
| `fix` | Runs all available auto-fixers (e.g., import optimization). | **Auto-Fix** |
| `ci-template` | Generates CI/CD pipeline templates (GitHub Actions, GitLab CI). | **CI/CD** |
